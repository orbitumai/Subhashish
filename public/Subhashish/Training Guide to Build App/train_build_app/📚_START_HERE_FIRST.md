## 🚨 Troubleshooting

### Guide doesn't load properly
**Fix:** Make sure all files are in same folder, open `index.html` in modern browser (Chrome, Firefox, Edge)

### Search doesn't work
**Fix:** Wait 5-10 seconds after page loads (needs to index), check internet for CDN

### Prompts don't work in Cursor
**Fix:** Make sure you copied the ENTIRE prompt including CONTEXT/TASK/CONSTRAINTS/ACCEPTANCE sections

### App won't deploy to Vercel
**Fix:** Check `deployment_commands.md` → Part 7, verify GitHub connection

### Database queries fail
**Fix:** Check `security_guardrails.md` → RLS policies must be enabled
