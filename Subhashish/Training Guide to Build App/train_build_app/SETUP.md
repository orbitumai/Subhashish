2. **Open `index.html`** in your web browser

## Tips for Best Experience

### For Reading
- Use **fullscreen** (F11 on Windows, Cmd+Ctrl+F on Mac)
- **Increase text size** if needed (Ctrl/Cmd + Plus)
- **Print to PDF** if you want offline backup (Ctrl/Cmd + P)

### For Building
- Keep this guide **open in one monitor**
- **Cursor/code editor in another**
- **Copy prompts directly** from guide into Cursor
- **Follow sequentially** - don't skip steps

## Troubleshooting

### Sidebar doesn't show
- Make sure `_sidebar.md` is in same folder as `index.html`
- Check browser console (F12) for errors

### Search doesn't work
- Wait for page to fully load (may take 5-10 seconds first time)
- Check internet connection (needs CDN for first load)

### Code blocks don't copy
- Click the "Copy" button in top-right of code block
- If missing, refresh page

### Page looks broken
- Try different browser (Chrome, Firefox, Edge)
- Clear browser cache (Ctrl/Cmd + Shift + R)
- Make sure JavaScript is enabled
